### 🔭 What I'm Building
 
**🛰️ [Meridian](https://github.com/viper9009adr/meridian)**
 
An open-source **multi-agent context transport system** built on MCP + gRPC + PostgreSQL with ZSTD binary compression. Meridian enables stateless session reconstruction — a fresh orchestrator can recover complete project state from two `fetch_context` calls. It uses parallel CTE graph traversal for context retrieval. Licensed Apache 2.0.
 
**🤖 [Self-BOT](https://github.com/Viper9009adr/Self-Bot-)**
 
An AI assistant built entirely with Claude + Meridian MCP as its backbone. It's a live showcase of what Meridian can do: cross-session memory, multi-tenant context injection, JWT-based auth, and a pre-LLM deterministic logic layer.
